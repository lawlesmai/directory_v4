This directory contains a recent revision of OS X/mac OS drivers for DisplayLink based USB Display Adapters.

These drivers are reserved for use in cases where the most up to date driver version are not working correctly with your computer.

We strongly recommend contacting our Technical Advisory Team before installing these older drivers. We recommend this because there may be other factors stopping the most up to date driver from working on your computer.

Please visit http://www.startech.com/Support to contact us for further support.
